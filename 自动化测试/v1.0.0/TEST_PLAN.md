# v1.0.0 Windows 正式发布测试计划（门禁待满足）

## 已可复用的本机门禁

- Unit / typecheck / 两轮 Electron E2E。
- packaged smoke、真实升级、数据保留、自启动。
- 安装包 SHA-256 与 Authenticode 签名状态核验。

## 发布前新增必测项

- RG-101：签名后的最终安装包证书链、时间戳、发行主体与 hash 一致。
- RG-102：无开发环境、独立 Windows 用户完成安装、引导、采集、计划、复盘和备份恢复。
- RG-103：用户理解“窗口标题采集、可暂停、隐私排除、ActivityWatch 原始数据边界、卸载与本地数据处理”。
- RG-104：外部用户发现的高严重度问题均完成复现、修复和回归。
- RG-105：最终签名包重新完成升级、数据保留和 Windows 自启动验证。

## 禁止替代

- 新 profile 自动化、开发机安装与构建成功不得替代 RG-102 至 RG-104。
- 未签名 installer 不得替代 RG-101。
