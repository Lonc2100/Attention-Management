# v0.7.3 · 跨午夜工作日边界

发布日期：2026-07-19

## 改进

- 凌晨仍在持续工作的时间继续归入前一工作日，不再被零点硬切开。
- 新工作日由主要休息后的首次返回开始；同一天后续白天长时间离开不会再次切日。
- 通宵活动保持在开始日，直到主要休息结束；ActivityWatch 暂时不可用时保留最近可信工作日。
- 活动明细新增工作日边界、判断来源、人工调整和恢复自动判断。
- 首页新增最近/最晚离开电脑时间，并明确它不等于真实入睡时间。
- 实时 AFK 状态与历史工作日归属分开计算，排除睡眠区间时仍会如实显示“已离开电脑”。

## 数据与隐私

- 存储 schema 从 v6 升级到 v7，保留原有记录、Codex 上下文、设置、纠错和规则。
- 工作日边界属于本地派生数据；人工调整可撤销，不改写 ActivityWatch 原始事件。
- 不录屏、不采集键盘正文，也不声称准确识别睡眠。

## 边界

- 主要休息默认阈值为 6 小时，但自动切日只取每个自然日的首次合格返回，不把白天每一次长离开都当成隔天。
- 仍只支持 Windows x64；安装包未代码签名，因此仍是本地 beta，不宣传为可信公开正式版。
