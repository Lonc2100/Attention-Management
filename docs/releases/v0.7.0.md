# v0.7.0 · 首页工作活动与安静悬浮胶囊

发布日期：2026-07-16

## 新增

- 首页新增近一年工作活动热力图，在每日、每周、每月三种粒度间切换。
- 每格使用固定强度区间，避免当天数据变化导致整张图颜色含义漂移。
- 每日格支持悬浮查看时长并点击进入当天活动明细；来源不可用与真实零活动明确区分。
- 新增本周期实际电脑投入、日均投入、成果注意力占比、45 分钟长专注块、优先成果完成和证据完整度。

## 优化

- 悬浮专注窗折叠态从 316×68 缩至 252×48，展开态从 322×214 缩至 268×150。
- 折叠态只保留状态点、项目、连续专注和展开操作，降低颜色、边框和按钮对比度。
- 保留不抢焦点、置顶/桌面模式、拖动位置记忆、隐藏恢复和展开摘要。
- 纳入 v0.6.2 之后尚未安装的主题 token、组件约束和编辑式界面规范。

## 统计口径

- “实际电脑投入”只计算 ActivityWatch 记录到的非 AFK 前台窗口时间，不等同于成果或效率。
- “成果注意力”只计算早间成果显式关联项目的确认时间，同一项目不会因关联多个成果而重复计数。
- “长专注块”指连续项目注意力达到 45 分钟的时间片。
- 优先成果完成只依据晚间复盘；证据完整度同时反映复盘和成果关联覆盖。

## 发布边界

- 仅支持 Windows x64。
- 安装包未代码签名，Windows 可能显示安全提醒。
- 本地优先，不录屏、不记录键盘正文；ActivityWatch 原始事件不会被修改。
