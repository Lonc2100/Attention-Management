# v1.0.0 发布验证设计

## 验证分层

1. **构建事实**：固定 commit、installer hash、第三方许可证、签名状态。
2. **本机回归**：Unit、typecheck、两轮 E2E、packaged smoke、升级、数据保留与自启动。
3. **独立外部验收**：真实 Windows 用户在非开发机上执行安装、引导、采集、计划、复盘、备份恢复和卸载/数据处理理解。
4. **发布事实**：签名验证、可信下载渠道、版本说明及反馈闭环。

任何下层通过都不能替代上层：本机 E2E 不替代外部验收，未签名包不替代可信发布。

## 隐私约束

- 外部测试只收集必要的结论、版本、错误复现步骤和用户自愿提供的截图。
- 不要求上传 ActivityWatch 原始事件、窗口标题、Codex 线程、路径、AI 文本或真实密钥。
