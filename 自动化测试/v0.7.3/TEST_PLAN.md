# v0.7.3 测试计划

## 自动化

- 工作日纯函数：跨午夜、主要休息、白天长离开、通宵、人工覆盖。
- ActivityWatch：无标题区间查询、31 日批次上限、范围守恒与断连降级。
- 存储迁移：v6 → v7，旧记录与设置不变，边界覆盖可增删。
- 聚合：项目、应用、AFK、Codex 上下文在工作日范围内守恒。
- UI：边界来源、最晚离开电脑、人工调整与撤销。

## E2E

1. 启动应用并确认首页工作日、记录状态和时间图可见。
2. 使用固定跨午夜事件验证时间轴与项目归属。
3. 设置人工边界并确认前后活动重新归属。
4. 撤销人工边界并确认恢复自动判断。
5. 连续执行两轮，确认无缓存污染。

- `npm test`、类型检查、生产构建。
- 打包 smoke。
- 真实升级安装、数据哈希、schema、自启动和采集恢复验证。
