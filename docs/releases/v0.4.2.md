# v0.4.2 — 悬浮预览性能优化

v0.4.2 采用固定锚点提示卡，解决首页圆环、图例和时间轴悬浮预览跟随鼠标时的卡顿。

## 变化

- 鼠标进入项目目标时提示卡只定位一次；在同一目标内部移动不再持续触发 React 状态更新。
- 提示卡优先出现在目标右侧，空间不足时自动选择左侧或上下方，并限制在窗口边界内。
- 圆环、图例和时间轴仍会联动高亮同一项目，键盘聚焦行为保持一致。
- 移除时间轴批量 `filter` 和提示卡 `backdrop-filter`，弱化效果改用透明度，卡片使用不透明深色背景。
- ActivityWatch 采集、Codex 项目归属、时间统计与本地数据格式均未改变。
