# v0.3.0 — 悬浮专注窗与注意力驾驶舱

v0.3.0 把已有的真实采集数据升级为可持续查看的注意力反馈系统。

## 新增

- 今日注意力圆环：全部项目与应用使用同一个有效电脑时间分母。
- Codex 项目叶子：同属 Codex 的项目使用同一绿色色系，但分别计时，父级不重复计入。
- 项目化时间轴：自动聚焦当天真实活动范围，呈现项目、应用、待分类和 AFK。
- 悬浮专注窗：默认置顶，显示当前项目与连续专注时间，支持展开、折叠、拖动、隐藏和恢复。
- 悬浮模式：可在设置中切换“默认置顶”和“停留桌面”。
- 多显示器回退：保存窗口坐标和屏幕；目标屏幕消失时回到主屏幕右上角。

## 调整

- 今日概览只呈现注意力数据；成果计划、晚间复盘、诊断和采集控制回到各自页面。
- 离开 Codex 后显示“最近确认”，但不继续把其他应用时间算入该项目。
- 设置数据从 v2 自动迁移到 v3，不改写 ActivityWatch 原始记录。

## 隐私与边界

- 仍然不录屏、不采集键盘正文。
- 其他应用不会自动强行归入最近 Codex 项目。
- 本版不包含活动明细纠错、手机、macOS、团队或计费能力。
