# v0.3.1 — 首屏分布与图表联动优化

v0.3.1 是 v0.3.0 的可用性增强版，不改变任何采集或项目归属规则。

## 本次更新

- 首页采用“分布优先”的紧凑布局，常用分辨率首屏可完整看到注意力圆环和时间轴。
- 当前项目与连续专注进入顶部命令栏；提醒与电脑活跃、Codex 注意力、项目覆盖压缩为摘要行。
- 圆环改为独立 SVG 扇段，圆环、图例、时间轴悬浮时会显示统一提示并联动高亮。
- 时间轴提示增加准确起止、时长和记录区间占比；窄时间块也可命中。
- 悬浮窗展开尺寸从 336×278 缩小为 322×214，增加今日项目占比并减少空白。
- 动效遵循系统“减少动态效果”设置；长名称会安全截断。

## 保持不变

- ActivityWatch 数据、AFK 规则、Codex 项目识别、本地存储、早间计划、晚间复盘和隐私边界均未修改。

## 验证

- 26 个单元测试通过（另有 2 个按环境跳过的集成测试）。
- TypeScript 类型检查与生产构建通过。
- 11 条 Electron 端到端链路通过，覆盖 1440×900 / 1600×1000 首屏、图表联动悬浮、悬浮窗生命周期、采集暂停恢复、Codex CLI 和重启持久化。
- 打包态烟测与安装态 Windows 自启动回归通过。
