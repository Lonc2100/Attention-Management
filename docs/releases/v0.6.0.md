# v0.6.0 — 本地 Beta 准备

v0.6.0 将产品从“开发者自己能使用”推进到“可进行受控 Windows 测试”：首次启动明确说明采集边界；隐私规则只清除本应用的派生展示；本地数据可备份、校验恢复并自动留下恢复点；CSV 和诊断导出不包含窗口标题、路径、线程 ID、项目键或 AI 文本。

## 交付内容

- 首次启动引导：只有 ActivityWatch 已连接且采集开启时才可确认开始。
- 隐私规则：应用名 + 窗口标题包含条件；对时间轴、应用/项目汇总、成果证据、个人规律、AI 摘要和活动明细生效，不写改 ActivityWatch 原始 bucket。
- 本地备份/恢复：备份 envelope 带格式与 schema；恢复前验证格式，恢复前自动复制一份本地恢复点。
- 聚合 CSV、脱敏诊断导出；原始 ActivityWatch 数据继续由其官方导出功能负责。
- 设置页明确：当前安装包未签名、不自动更新，不应被当作可信公开发布。
