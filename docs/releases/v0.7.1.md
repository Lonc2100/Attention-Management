# v0.7.1 · 首页信息层级收敛

发布日期：2026-07-16

## 首页减法

- 删除重复日期与大标题、三张顶部次要统计卡和独立成果说明卡。
- 将当前项目、连续专注、今日投入、优先成果及采集状态合并为约 60px 的单条状态栏。
- 早间计划或晚间复盘待办保留在优先成果区域中，不再额外占用一整行。

## 数据排布

- 工作活动标题压为单行，统计口径改为可访问的信息提示。
- 首页移除六张周期解释卡，保留每日、每周、每月热力图切换与悬浮证据。
- 1440px 及以上常用宽度中，工作活动热力图与今日注意力圆环并排展示。
- 缩小每日热力格，完整年度视图不再出现横向滚动条。

## 验证

- 单元测试：61 项通过，3 项需显式集成环境的测试跳过。
- Electron E2E：17 项通过；覆盖 1440×900、1600×1000、1920×1080 三种窗口。
- 验证紧凑状态栏不超过 64px、首页无横向溢出、宽屏图表共用一行。

## 边界

- 本版只调整首页信息层级与布局，不改变 ActivityWatch 采集、AFK、项目归属、成果证据或本地存储口径。
- 仅支持 Windows x64；安装包仍未代码签名。
